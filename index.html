Tutorial Hosting Gratis di GitHub Pages

1. Apa itu GitHub Pages?

GitHub Pages adalah layanan gratis dari GitHub untuk hosting website statis. Sangat cocok untuk portofolio, blog, dokumentasi, atau projek web sederhana.

2. Syarat Hosting di GitHub Pages

  • Punya akun GitHub
  • Website dalam format statis (HTML, CSS, JS)
  • Koneksi internet untuk upload ke GitHub

3. Langkah-langkah Hosting

a. Buat Akun GitHub

Kunjungi https://github.com dan daftar akun gratis.

b. Buat Repository Baru

  1. Login ke GitHub
  2. Klik tombol New atau + ‚Üí pilih New repository
  3. Isi nama repo (misalnya my-website)
  4. Pilih Public
  5. Centang ‚ÄúAdd a README file‚Äù (opsional)
  6. Klik Create repository

c. Upload File Website

Siapkan file seperti index.html, style.css, dsb. Lalu upload:

  • Via GitHub Web: klik Add file ‚Üí Upload files
  • Atau via Git:
    • git clone https://github.com/username/my-website.git
cd my-website
# tambahkan file
git add .
git commit -m "Upload website"
git push origin main

d. Aktifkan GitHub Pages

  1. Masuk ke repo kamu
  2. Klik tab Settings
  3. Scroll ke bawah, pilih Pages
  4. Pilih source: main dan folder /(root)
  5. Klik Save
Tunggu beberapa detik, website kamu akan tersedia di:
https://username.github.io/nama-repo/

4. Tips Tambahan

  • Pastikan file utama bernama index.html
  • Gunakan folder / saat memilih source Pages
  • Bisa dipakai untuk React, Vue, atau proyek static build
  • Custom domain? Tambahkan file CNAME

5. Penutup

Dengan GitHub Pages, kamu bisa membuat website profesional tanpa biaya hosting. Mudah digunakan, cepat, dan cocok untuk siapa saja.

6. FAQ (Pertanyaan Umum)

üîπ Bagaimana cara menambahkan file index.html langsung dari GitHub tanpa Visual Studio Code?

Kamu bisa menambahkan file langsung dari web GitHub tanpa perlu aplikasi tambahan. Caranya:

  1. Buka halaman repository kamu di GitHub
  2. Klik tombol Add file ‚Üí Create new file
  3. Di kolom File name, ketikkan: index.html
  4. Tulis isi HTML sederhana di bawahnya, misalnya: 
    <h1>Hai, Nama saya R Hendro Jaya Malau!</h1>
  5. Scroll ke bawah, isi pesan commit misalnya: "menambahkan index.html"
  6. Klik tombol Commit new file

Setelah itu kamu bisa mengaktifkan GitHub Pages seperti biasa. File index.html akan menjadi halaman utama website kamu.
